# ♛ Шашки - Игра онлайн

Игра в шашки с возможностью игры 1 на 1 или против бота с разными уровнями сложности.

## 🎮 Функции

- 👥 Режим 1 на 1 (игра с другом)
- 🤖 Режим против бота (4 уровня сложности)
- 🎯 Обязательное взятие (можно включить/выключить)
- 👑 Дамки с возможностью хода назад
- 🔄 Отмена хода
- 🎨 Выбор цвета (белые/чёрные)

## 🚀 Как играть

1. Выберите режим игры
2. Нажмите на шашку для выбора
3. Нажмите на клетку для хода
4. Наслаждайтесь игрой!

## 📱 QR-код

Сканируйте QR-код для быстрого доступа к игре на телефоне:
- Перейдите на страницу `qr.html`
- Или нажмите "QR-код для входа" в игре
